## 關鍵教訓

- **偏鄉教育不能寫成熱血教師救援**：只寫一個好老師，會把結構問題縮成個人勵志
- **逐字稿最有價值的是結構，不是字面**：錯字很多沒關係，能萃出四層視角就夠了
- **法律條文本身就是敘事證據**：條文越細，越能反證現場困難不是抽象概念
- **build 失敗要先分流**：先確認是 Node/Astro 環境卡住，別把工具鍋甩給文章
