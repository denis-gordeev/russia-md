# Contributing

Thanks for contributing to `Russia.md`.

## Scope

This repository is a structured knowledge base about Russia. Good contributions are narrow, sourced, and easy to review.

## What To Add

- New articles in `russia-knowledge/`
- Better sources and citations
- Russia-specific datasets and charts
- UI, SEO, and deployment improvements
- Agent integration docs in `SKILLS.md`

## Writing Rules

- Prefer concise, factual prose over slogans.
- Cite primary or clearly attributable sources.
- Preserve a neutral, readable tone.
- Explain why the topic matters, not only what it is.

## File Layout

- `russia-knowledge/` is the active article corpus.
- `src/` contains Astro pages and site shell code.
- `legacy-pages/` and `legacy-content/` are archived material from earlier iterations and should not be extended for current work.

## Workflow

1. Create a branch.
2. Make a focused change.
3. Run `npm run build`.
4. Open a pull request with a short summary and any source notes.
